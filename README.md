## **Project Overview**

Solo Barbershop is a **modern, professional haircut-focused barbershop website**. The goal of this project is to build a clean, responsive, and user-friendly web application to showcase **haircut services**, gallery, and contact information.  

This project is part of the **Future Interns Full Stack Web Development Internship** (Task 3 – Local Business Website & Live Pitch).

---

## **Features**

- Fully responsive layout (desktop, tablet, mobile)
- Hero section with tagline and call-to-action
- Services section (Classic Haircut, Fade Haircut, Line-Up, Kids’ Haircut)
- Gallery section showcasing haircut examples
- About section with business description
- Contact form powered by [Formspree](https://formspree.io/) (no backend required)
- Smooth scrolling navigation
- Modern hover effects and interactive buttons
- SEO-friendly HTML structure
- Mobile-first design

---

## **Technologies Used**

- HTML5  
- CSS3 (Flexbox & Grid for layout)  
- JavaScript (Smooth scrolling navigation)  
- Google Fonts (Roboto)  
- Formspree (for contact form)  
- Netlify (deployment)
